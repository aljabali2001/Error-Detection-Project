# Error Detection & Correction Project

## Description
This project demonstrates error detection and correction techniques
using a UDP client-server-client architecture in Java.

## Architecture
Client1 → Server (error injection) → Client2

## Implemented Methods
- Even Parity
- Two-Dimensional Parity
- CRC-16
- Internet Checksum
- Hamming (7,4) Error Correction

## How to Run
1. Run Server
2. Run Client2
3. Run Client1
